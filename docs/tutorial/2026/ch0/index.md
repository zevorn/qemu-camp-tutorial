# 导学阶段

为了帮你更顺畅地入门 QEMU，我们专门准备了导学阶段的内容。在这里，你能通过一系列教程，系统掌握体系结构、编程和模拟器相关的基础知识。只要你有较强的自学能力和主动性，就能很快上手，顺利进入下一阶段的学习。

> PS：导学阶段的学习资料，同时可作为后续阶段的补充学习资源，你可根据自身知识掌握程度灵活规划学习节奏。
