# 基础阶段

基础阶段的目标很明确：带你直观认识 QEMU 用到的核心软件技术，吃透 QEMU 的关键概念、基本用法和调试技巧。另外，我们会通过编程练习帮你打牢 C 和 Rust 的基础，这些知识对后面的学习特别有用。

这一阶段我们会一起学习这些内容：

!!! tip "概览"

    - 虚拟机的发展历史与基本工作原理
    - QEMU 的常用参数与启动流程
    - QEMU 的核心概念：QOM、MemoryRegion 等
    - QEMU 常用的调试手段
