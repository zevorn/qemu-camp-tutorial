# 聚焦开源与云原生！GTOC&CNB 首次线下 Meetup 完美收官

!!! note "主要贡献者"

    - 通讯：[@zevorn](https://github.com/zevorn)
    - 摄影：[@youkun](https://cnb.cool/u/youkun)

近日，格维开源社区（GTOC）联合 CNB 首次线下 Meetup 活动于 2026 年 1 月 31 日（周六）在腾讯上海滨江大厦 411&412 会议室圆满举办。本次活动以“开源技术与云原生实践”为核心，秉持“格物致知、多维创新、技术包容、社区开放”的理念，汇聚了众多技术开发者、开源爱好者及行业同仁，共同打造了一场兼具干货分享与深度互动的技术交流盛会。

作为 GTOC 与 CNB 联合推出的首场线下活动，本次 Meetup 精心规划了多元议程，涵盖主题分享、自由交流、合影打卡等多个环节，旨在打破线上沟通壁垒，搭建真实高效的技术对接平台。活动当天 13:30 起，参会嘉宾陆续签到入场，在“面基时间”中初步结识、交流行业动态；14:00 活动正式拉开帷幕，CNB 上海分舵主轩率先开启破冰环节，为现场氛围注入活力。

随后的主题分享环节亮点纷呈，多位行业资深人士与社区核心成员带来了极具价值的实战经验与技术洞察。格维开源社区负责人泽文亲临现场，围绕“QEMU 训练营 2026 在云原生场景下的最佳实践”展开深度分享，详细拆解了训练营的实训规划与云原生场景的融合路径，为开发者解锁虚拟化技术学习的实用方案。

![泽文](../../image/cnb-and-gtoc-meetup-01.jpg){ style="width: 100%; height: auto;" }

社区核心成员喻天宇聚焦联邦学习 FedSWA 等开源项目，深入浅出地解析了其在 CNB 平台上的复现思路与实战技巧，为参会者提供了可落地的技术参考。

![喻天宇](../../image/cnb-and-gtoc-meetup-02.jpg){ style="width: 100%; height: auto;" }

此外，知识库大师带来“DIY 一个更懂你的小爱音箱”实操分享，开源文化爱好者二丫讲解“云原生开发打破本地资源限制”的核心逻辑，TDP 宣传大使 Ar-Sr-Na 分享“让创意安稳落地的开发心法”，TDP 社区 ZhienXuan 介绍“萌新如何用 GLM 开发 Linux 项目”，开源文化爱好者宵伯特解读“搭建云端 Agent 农场”的关键技术，多维度、多层次的分享内容满足了不同阶段开发者的学习需求，引发了现场阵阵共鸣。

活动期间，主办方特别设置了茶歇自由交流时段，参会嘉宾纷纷围绕技术难点、项目合作、行业趋势等话题展开热烈讨论，积极拓展人脉资源、交换项目经验，现场互动氛围浓厚。17:00 后，活动进入交流合影环节，CNB 产品团队 youkun 与春雨牵头组织，参会者共同打卡大合照，定格此次线下相聚的珍贵瞬间。此外，所有到场嘉宾均领取了“上海分舵”专属特权，关注社区公众号后还可免费使用 AI Infra/QEMU/Compiler/Linux 等方向的 ima 知识库，为后续技术学习与实践提供了有力支持。

本次 GTOC&CNB 首次线下 Meetup 的成功举办，不仅为技术开发者们提供了一个学习前沿技术、交流实战经验的优质平台，更推动了开源社区与云原生领域的资源整合与生态共建。未来，GTOC 与 CNB 将持续秉持开放共享的理念，推出更多形式丰富、内容扎实的线下交流活动，助力技术同仁们深化合作、共同成长，为开源技术与云原生行业的创新发展注入更多活力。
