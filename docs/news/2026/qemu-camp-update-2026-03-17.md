## 2026 年度课程全面升级

相较往期，本次训练营在课程和项目深度和广度上均有大幅提升，以下是几大核心亮点：

**GPGPU 建模专题（全新）**

新增 GPU 体系结构与建模系列课程，从 SIMT 执行模型到 PCIe 设备实现，从 DMA/MSI-X 到 QTest 自动化测试，带你在 QEMU 中完整实现一个 GPGPU 设备原型。实验阶段还设有开放题目——设计简单 AI 软件栈（CUDA 风格）、集成 Vortex simx 并适配 ArceOS/rCore 等操作系统训练营项目。

**AI 硬件加速建模（全新）**

基于 CXL 协议的 AI 加速卡建模，紧跟异构计算技术前沿。项目集成了 [CXLMemSim][cxlmemsim-link] 与 [hetGPU][hetgpu-link] 两大开源方案：通过 QEMU 模拟 CXL Type-2 加速器设备，Guest 端 CUDA 应用无需改动，经 libcuda shim 将 CUDA Driver API 翻译为 CXL 命令，由 hetGPU 后端执行命令处理与设备内存仿真，实现端到端的跨平台 GPGPU 仿真体验。表现优秀的学员可以参与相关的科研项目。

**AI 大模型自动化建模（全新）**

引入大模型 Agent 驱动的自动化外设建模机制，以硬件参考手册、数据手册及开源驱动代码为输入，实现寄存器结构自动抽取、行为语义推断、中断模型生成与模型代码自动输出，突破传统手工建模的效率瓶颈，面向 STM32 等主流 MCU 实现复杂外设的规模化建模。

**数字星务计算机建模（升级）**

基于 RISC-V 架构的勘智 K230 芯片，对卫星星务计算单元进行 QEMU 硬件建模。相比第一期的基础建模，本期结合了真实的卫星应用场景，从 K230 基本外设支持出发，逐步推进到星务系统安全模块的完整运行，探索开源 RISC-V 芯片在航天领域的落地实践。

**二进制动态翻译深度解析（升级）**

专项讲解 TCG 工作原理，揭示 QEMU 核心引擎的运行机制，并且在项目阶段提供了 Wine-CE 跨平台转译的探索。

**Rust for QEMU（升级）**

新增 Rust FFI、SPI Bus、GPIO 等多方向实验，共计 10 道实验题目，C 与 Rust 双语言并行教学。

## "三二五六" 四阶段课程体系

整个训练营为期 16 周，划分为四个阶段：

| 阶段 | 时长 | 内容 |
| --- | --- | --- |
| 导学阶段 | 3 周 | C/Rust 编程基础、RISC-V 体系结构，GPGPU 体系结构、QEMU 环境搭建 |
| 基础阶段 | 2 周 | 虚拟机工作原理、QEMU 启动流程、QOM、MemoryRegion、调试技巧 |
| 专业阶段 | 5 周 | CPU 建模、TCG 翻译、硬件 MMU、中断异常、主板与外设建模、GPGPU、PCIe 模拟、Rust 建模 |
| 项目阶段 | 6 周 | K230 数字星务计算机建模、CXL AI 加速卡、大模型外设建模、Wine CE 跨平台转译 |

课程共包含 20+ 篇系统讲义，覆盖虚拟化技术从入门到实战的完整链路。

## 实验方向，多轨并行

训练营提供多条并行的实验路径，学员可根据兴趣自主选择：

- **基础阶段：C 编程实验** —— 夯实系统编程基础
- **基础阶段：Rust 编程实验** —— 拥抱 QEMU 的 Rust 化未来
- **专业阶段：CPU 建模实验** —— 基于 RISC-V Xg233ai 处理器，配套完整数据手册与实验手册
- **专业阶段：SoC 建模实验** —— 基于 G233 SoC，从外设到总线的完整建模实践
- **专业阶段：GPGPU 建模实验** —— GPU 功能建模，本期重点方向，基于 QTest 框架自动化评分
- **专业阶段：Rust 建模实验** —— I2C Bus、SPI Bus、GPIO 全覆盖

## 关于我们

QEMU 训练营是在清华大学陈渝老师和李明老师的倡议下发起的公益性技术训练营，致力于搭建虚拟化与模拟器技术的学习与实践平台，助力开发者深入掌握 QEMU 核心能力，推动开源生态发展。训练营课程内容以 CC BY-SA 4.0 协议开源，代码以 MIT 协议开源。

### 主办单位

**格维开源社区（GTOC）**
训练营发起组织，秉持"格物致知、多维创新、技术包容、社区开放"的理念，聚焦虚拟化、编译器、操作系统等系统软件方向，持续推动开源技术的学习、实践与社区共建。
[了解更多 >>][gevico-link]

**华中科技大学开放原子开源俱乐部**
于 2023 年 6 月正式授牌成立，是一个致力于促进开源文化发展、贡献优秀开源软件的活跃开源社区，遵循"开放、共享、协同、贡献"的开源精神。俱乐部由来自各专业领域的师生组成，核心关注 Linux 内核、国产 Linux 发行版、物联网操作系统、RISC-V 等方向，运营有华中科技大学开源镜像站（HUST Mirror）、HCTT 翻译团队、Linux 内核揭秘等多个知名开源项目，定期举办技术讨论班与圆桌会议。
